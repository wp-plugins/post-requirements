This plugin allows you to require posts to have thumbnail set and/or tags.

== Description ==

If your site is depending on posts to have their thumbnail images set, this is the plugin for you. Also makes it possible
to require tags in every new post.

When trying to save a post without at thumbnail or tags and when activated, the user will get a warning and the post will
be saved as a draft instead.

== Installation ==

1. install it
2. activate it
3. check out the settings page

== Frequently Asked Questions ==

= Can you require _just_ tags or _just_ thumbnails? =

Yes. You can choose which of the two (or both) you want to require in the options page of the plugin.
